# 👋 Sobre mim

Sou **Engenheiro de Software** com experiência no desenvolvimento de soluções completas, participando de todas as etapas do ciclo de desenvolvimento: arquitetura, modelagem de banco de dados, backend, integrações, aplicações desktop, web e mobile.

Atualmente atuo no desenvolvimento de sistemas corporativos utilizando **Delphi**, além de desenvolver aplicações em **Flutter** e plataformas SaaS voltadas para diferentes segmentos.

Tenho paixão por criar softwares robustos, escaláveis e de fácil manutenção, sempre buscando aplicar boas práticas de engenharia de software e arquitetura de sistemas.
